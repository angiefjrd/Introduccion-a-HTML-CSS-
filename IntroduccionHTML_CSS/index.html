HTML (Hipertext Markup Language)

Vamos a empezar escribiendo un archivo de HTML simple!

                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initil-scale=1.0">
                        <head/>
                        <body>
                        <h1>Hola Mundo!</h1>
                        </body>
                        </html>

Nota: Siempre guarda tu archivo con la extension .html y abrelo en tu navegador para verlo funcionar.

EL codigo renderizado se mostraria en el navegador asi:

Hola Mundo!

Insertando Imagenes

                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initil-scale=1.0">
                        <head/>
                        <body>
                            <img src=https://via.placeholder.com/200 alt="Descripcion de la imagen">
                        </body>
                        </html>
