HTML (Hipertext Markup Language)

Vamos a empezar escribiendo un archivo de HTML simple!

                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initil-scale=1.0">
                        <head/>
                        <body>
                        <h1>Hola Mundo!</h1>
                        </body>
                        </html>

Nota: Siempre guarda tu archivo con la extension .html y abrelo en tu navegador para verlo funcionar.

EL codigo renderizado se mostraria en el navegador asi:

Hola Mundo!

Insertando Imagenes

                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initil-scale=1.0">
                        <head/>
                        <body>
                            <img src=https://via.placeholder.com/200 alt="Descripcion de la imagen">
                        </body>
                        </html>

Nota: podemos ingresar imagenes de manera local o mediante url.

Sintaxis para insertar un formulario.

                        <!Doctype hml>   
                        <html lang="en"> 
                        <head> 
                        <body> 
                        <form> 
                            <input type="text" placeholder="First Name" name="first"> 
                            <input type="password" placeholder="Password" name="password"> 
                            <div> 
                                <Favorite Color: Blue> 
                                <input name="color" type="radio" value="green"> Green> 
                                <input name="color" type="radio" value="yellow"> Yellow> 
                            <input type="submit">
                        </form>
                        </body>
                        </html>

EL codigo renderizado se mostraria en el navegador asi:

Favorite Color: Blue Green Yellow Red

Sintaxis de la etiqueta header

                        <body>
                        <header></header>
                        </body>

Sintaxis de la etiqueta nav

                        <body>
                        <header>
                        <nav></nav>
                        </header>
                        </body>

Podemos encontrar diferentes nav en una misma pagina web

Por ejemplo una lista ordenada con las secciones de una pagina.

                        <body>
                            <header>
                                <nav>
                                    <li>
                                        <a href="./Inicio.html">Inicio>
                                    </li>
                                    <li>
                                        <a href="./Productos.html">Productos>
                                    </li>
                                </nav>
                            </header>
                        </body>
